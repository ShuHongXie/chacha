### 斜率和截距 y=kx+b

1. 斜率：k，k 越大，直线陡峭

### 导数

1. 定义：类似于斜率，只不过描述的是曲线变化的过程，**某一点的导数等于该 点的切线斜率**
2. 判断方式：下面公式的极限存在时，说明在该点处可导
3. 公式：
   - f'(x) = lim (x->x0) [f(x)-f(x0)]/x-x0
   - f'(x) = lim (x->0) [f(x0+△x)-f(x0)]/△x
4. 写作形式
   - y'|x=x0
   - f'(x0)
   - [dy/dx]|x=x0
5. 单侧导数：左导和右导
6. 函数在某点可导的充要条件：左右斜率相等，即 f'(x0)= A <=> F'\_(x0) = f'+(x0)=A

### 可导与连续

1. 可导与连续：
   - 连续：不间断
   - 可导: **光滑**
2. 可导与连续的关系：可导 => 连续
3. 常用导数

### 微分

3. 微分四则运算：1. d(u+-v) = du +- dv

5. 隐函数：y=y(x)由方程 F(x,y)=0 所确定，求 dy/dx
   - 方法：dy/dx = -Fx/Fy (注：dx/dy=-Fy/Fx)
   - Fx：F 对 x 的偏导，Fy：F 对 y 的偏导

```js
/**
 * 问题：函数y=y(x)由x+y+xy=1所确定，求dy/dx
 * 解：1.令F(x,y) = x+y+xy-1
 * 2. Fx = 1+y, Fy = 1+x
 * 3. dy/dx = -Fx/Fy = -(1+y)/1+x
 */

/**
 * 求由方程xylny+y=e^2x所确定的隐函数在x=0处的导数(dy/dx)|x=0
 * 解：将x=0代入原方程得y=1
 * 1. 令F(x,y) = xylny+y-e^2x
 * 2. Fx= ylny-2e^2x ,Fy = xlny+x+1
 * 3. dy/dx|x=0 = -Fx/Fy = 2
 */
```

### 微分中值定理

1. 罗尔定理：函数 f(x)满足 1. 在闭区间[a,b]上连续，2.在开区间(a,b)内可导，3.f(a) = f(b)，则至少存在一点 ξ∈(a,b)，使得 f'(ξ)=0
2. 拉格朗日中值定理：如果函数 f(x)满足 1. 在闭区间[a,b]上连续，2.在开区间(a,b)内可导，则存在一点 ξ∈(a,b)，使得 f'(ξ)=[f(b)-f(a)]/b-a
   - 曲线上存在某个点，使得该点的斜率等于端点的切线斜率，即平行

### 函数的单调性，极值与最值

1. 单调性：
   - 判定方法：f'(x) > 0，则 f(x)单调递增，反之单调递减
2. 极值
   - 定义：
     1. f(x) < f(x0),则 x=x0 为极大值点，f(x0)为极大值
     2. f(x) > f(x0)，则 x=x0 为极小值点，f(x0)为极小值
   - 第一判定定理：
     1. x < x0 时，f'(x) > 0; x > x0，f'(x) < 0， 则 x=x0 为极大值点；
     2. x < x0 时，f'(x) < 0; x > x0，f'(x) > 0， 则 x=x0 为极小值点；
   - 第二判断原理
     f'(x0) = 0 时，f''(x0) >0，则 x=x0 为极小值点(凹的)，f''(x0) <0，则 x=x0 为极大值点(凸的)
3. 驻点：若 f'(x0) = 0，则 x=x0 为 f(x)的驻点。
4. 若 x= x0 为 f(x)的极值点，则 f'(x0) = 0 或 f'(x)不存在
5. 最值
6. 凹凸性：
   - 凹曲线：曲线上任一点的切线总位于曲线下方 f''(x) > 0 为凹，反之为凸区间
7. 拐点：凹凸性的分界点称为拐点，记作(x0,y0)，拐点两侧 f''(x)必然异号
   - 若点(x0,y0)是曲线 y=f(x)的拐点，则 f''(x0) = 0 或 f''(x)不存在
8. 水平渐近线：若 lim(x->∞) f(x) = C，则称 y=C 是 y=f(x)的一条水平渐近线
9. 垂直渐近线：若 lim(x->x0) f(x) = ∞，则称 x=x0 是 y=f(x)的一条垂直渐近线

### 考点

1. 罗尔定理证明题：
   - 步骤：1. 构造函数 f(x)，2.验证 3 个条件 3.由罗尔定理可知，至少存在一点 ξ∈(a,b)，使得 f'(ξ)=0
2. 拉格朗日中值定理证明题：1. 构造函数 f(x)，2.验证 3 个条件 3. 由拉格朗日中值定理可知，下结论，对结论式子变形
3. 求单调性步骤
   1. 求 f(x)定义域
   2. 求出 f'(x) = 0 和 f'(x)不存在的点，将定义域划分若干个子区间
   3. 列表，根据 f'(x)在子区间内的符号，确定单调性
4. 求极值点和极值的步骤
   1. 确定 f(x)定义域
   2. 求出 f'(x) = 0 和 f'(x)不存在的点
   3. 列表
5. 求最值的步骤
   1. 求出 f'(x) = 0 和 f'(x)不存在的点
   2. 求出 1 中所有点的函数值和断点处的函数值
   3. 最大值 = max[极值，端点值]，最小值= min[极值，端点值]
6. 凹凸性与拐点的求解步骤
   1. 确定 f(x)定义域
   2. 求出 f''(x) = 0 和 f''(x)不存在的点
   3. 列表，由 f''(x)符号得出凹凸区间，凹凸区间的分界点即为拐点
7. 求渐近线
8. 利用单调性证明不等式和根的存在性之不等式的证明步骤
   1. 构造函数 f(x)
   2. 求导判断单调性
   3. 大于最低点，小于最高点

```js
/**
 * 例1：证明x<e^x-1，其中x>0
 * 证明
 * 1. 令f(x) = x-e^x+1
 * 2. f'(x) = 1-ex，当x>0时，f'(x) <0，单调递减
 * 3. f(x) < f(0)，即x<e^x-1，其中x>0
 */
```

9. 利用单调性证明不等式和根的存在性之唯一根的证明步骤
   1. 利用零点或罗尔定义证明至少由一个根
   2. 求导判断函数单调，得唯一根

```js
/**
 * 例1：证明方程x^3-3x^2+6x-1=0在区间(0,1)内有唯一实根
 * 证明
 * 1. 令f(x) = x^3-3x^2+6x-1
 * 2. 显然f(x)在[0,1]上连续 因为f(0) = 1, f(1) = -3 , f(0)*f(1)<0
 * 由零点定理可知，至少存在一点ξ∈(0,1)使得f(ξ) = 0
 * 3. 因为f'(x) = 3[(1+x)^2+1] > 0，f(x)在(0,1)上单调递增，综上所述，方程x^3-3x^2+6x-1在区间(0,1)内有唯一实根
 */
```

10. 恒等式的证明的步骤
    1. 构造函数 f(x)
    2. 求导验证 f'(x) =0
    3. f(x)=f(x0)=C

```js
/**
 * 证明：当|x|<=1时，恒有arcsinx+arccosx=π/2
 * 1.令f(x)=arcsinx+arccosx
 * 2.f'(x) = 1/根号(1-x^2)- 1/根号(1-x^2) =0
 * 所以f(x)为常值函数，即f(x)=C
 * 3.f(x)=f(0)= arcsin0+arccos0 = π/2
 * 即|x|<=1时，恒有arcsinx+arccosx=π/2
 */
```
