### 如何去绝对值？

1. 关键看绝对值(||)内部是大于 0 还是小于 0。大于 0，去绝对值，小于 0，去绝对值加负号(整体)。

```js
// 例题：1. y=|x-2|怎么去绝对值？
/* 
1. 当x-2>0，即x>2时 y=x-2
2. 当x-2<0，即x<2时，y=-(x-2)=x+2
y=|x-2| = {
  x-2，x>2
  2-x，x<2 
}
*/
```
