```js
// 例1-3
R1 = 1，R2 = 2，R3 = 10， U(s1) = 3V U(s2) = 1V，求电阻R1两端的电压U1?

// U = RI
// Us1 = U3+U1  U1 = U2+Us2
// I1 = U1/R1 = U1
// I2 = (U1-Us2)/R2
// I3 =  (Us1-U1)/R3
// 由KCL可得 I3 = I2+I1，设U1 = x,即 [(3-x)/10] = x + [(x-1)/2]，解得x=0.5V
```
